REQUISITOS FUNCIONALES

	- Situar la carpeta del proyecto en el escritorio del PC, bajo la siguiente ruta: 'C:\Users\Usuario\Desktop\'

	- Situar la aplicacion de Google Drive Descargable en la ruta de usuario, e incluir una carpeta en la nube llamada 'iSecurityCS'. Se corresponde a la siguiente ruta: 'C:\Users\Usuario\Google Drive\iSecurityCS\'.



FUNCIONAMIENTO


	Subir Archivo:

		- Situar el archivo a subir en el la carpeta con el nombre de 'Uploads'.
		- Ejecutar la aplicacion.
		- Introducir el nombre de dicho archivo y una password secreta, en el panel de 'Subir Archivo'.
		- Pulsar el boton de 'Subir'.


	Descargar archivo:

		- Ejecutar la aplicacion si no estaba ya ejecutada.
		- Introducir el nombre del archivo a descargar alojado en Google Drive. Normalmente el formato del archivo sera el siguiente: nombre.extension.txt.
		- Introducir la password correspondiente, es decir, la que se uso para subir dicho archivo.
		- Pulsar el boton 'Descargar'. Ir a la carpeta de Downloads para recoger el archivo.
